# Volatility Forecasting & Price Prediction

**Objective:**  
Forecast volatility and price movements using time series and ML models.

**Techniques:**  
- ARIMA / GARCH models  
- Random Forest / Gradient Boosting  
- LSTM (Python)  
- Rolling window cross-validation

**Tools:**  
R, Python, statsmodels, keras

**Contents:**  
- `forecasting_pipeline.R` / `.py` → model scripts  
- Plots: predicted vs actual, residual diagnostics  
- README.md → project summary  

**Outcome:**  
Built modular pipeline; demonstrated ML models can outperform classical methods in certain regimes.
