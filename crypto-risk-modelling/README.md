# Cryptocurrency Risk Modelling

**Objective:**  
Quantify the loss and gain behaviour of top 3 cryptocurrencies using statistical distribution fitting.

**Techniques:**  
- Fit 22 probability distributions  
- Compare using AIC/BIC, KS test  
- Tail-risk analysis  
- QQ-plots & goodness-of-fit diagnostics

**Tools:**  
R, fitdistrplus, ggplot2, actuar

**Contents:**  
- `crypto_analysis.R` → code  
- Plots: distribution fits & tails  
- README.md → project summary  

**Outcome:**  
Identified best-fitting distributions per asset and quantified tail-risk.
